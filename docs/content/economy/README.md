# economy

The economic model in plain English. This is the "why it works" section. It explains how
Pred makes money, how it stays neutral on outcomes, and how it stays solvent when leveraged
users lose, without any heavy math.

## Pages

- `overview.mdx`        The whole model in one read.
- `two-pots.mdx`        The two pools of capital and what each one backs.
- `betting-machine.mdx` The house engine: how bets are priced and matched.
- `lending-machine.mdx` The leverage engine: how borrowing and liquidation work.
- `safety-wall.mdx`     The soft ceiling and capital rule that cap the worst case.

The exact formulas behind these ideas live in `../technical`.
