# ui

Small, reusable building blocks for the interface, in the shadcn/ui style. These are generic
and know nothing about Pred. The app composes them into screens.

Examples include the button, input, slider, separator, badge, tooltip, table, dialog,
dropdown menu, and tabs. The tooltip here is themed to match the dark app and is used by the
trading hints.

If you need a new primitive, add it here so it stays consistent and reusable.
